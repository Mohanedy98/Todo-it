## Uses

- [x] OOP Concepts
- [x] Delegate Design Pattern
- [x] CoreData (Data Types, Relations, Concepts ..)(Core data version is available on CoreDataVersion Branch)
- [x] Realme (Realm version is available on master branch)

- iOS 10.0+
